Projet - Automatisation du traitement des données dans un tableur

  • BENOIT Nathaël
    • ⋅
  • BRUNET--LARE Rafaël

Décembre 2025
Excel, VBA, Google Apps Script
Lors de ce projet, nous avons créé une application automatisant la collecte et l'analyse de données de satisfaction sur la formation sur de Google Sheets et Excel.

Ce projet m'a permis de développer mes compétences en programmation VBA et Google Apps Script, ainsi que ma capacité à travailler en équipe pour atteindre un objectif commun.

Organisation du projet

Nous avons créé des formulaires HTML et VBA pour collecter les données de satisfaction des étudiants. Un tableau de bord a été créé pour visualiser les résultats de l'analyse des données, permettant aux formateurs de prendre des décisions éclairées pour améliorer la qualité de la formation.

Projet – Création d'une Base de Données de Gestion d’une Clinique Dentaire

  • BENOIT Nathaël
    • .
  • BABIN Paul

Décembre 2025
SQL
Lors de ce projet, nous avons créé une base de données pour gérer les informations d'une clinique dentaire, en utilisant SQL pour structurer les données.

Ce projet universitaire m'a permis de développer mes compétences en modélisation de données et en SQL, ainsi que ma capacité à travailler en équipe pour atteindre un objectif commun.

Organisation du projet

Lors de ce projet universitaire, nous avons commencé par analyser les besoins de la clinique dentaire en termes de gestion des données. Nous avons ensuite implémenté la base de données en utilisant SQL, en créant les tables et en définissant les contraintes d'intégrité. Enfin, nous avons créer des requêtes afin de visualiser les données de la clinique dentaire.

Projet Universitaire - Description et prévision de données temporelles

  • BENOIT Nathaël
    • .
  • CARRACO Camille

Novembre 2025
Excel / Word
Lors de ce projet, nous avons analysé des données temporelles liées aux fluctuations boursières de l'uranium.

Ce projet universitaire m'a permis de développer mes compétences en analyse de séries temporelles, en modélisation statistique et en utilisation d'outils tels qu'Excel pour traiter et visualiser les données.

Organisation du projet

Lors de notre projet universitaire, nous avons commencé par collecter les données temporelles sur les prix de l'uranium à partir de sources fiables. Nous avons ensuite nettoyé et préparé les données pour l'analyse, en traitant les valeurs manquantes et en identifiant les tendances saisonnières.

Nous avons par la suite réalisé plusieurs models afin de trouver le meilleure modèle de prévision pour les données de l'uranium.

Nous avons enfin rédigé un rapport détaillant nos méthodes, nos résultats et nos conclusions sur l'analyse des données temporelles.

    • .
  • CARRACO Camille

Php / Python / HTML / CSS / JavaScript

Projet - Site web dynamique

  • BENOIT Nathaël
    • .
  • CARRACO Camille
    • .
  • CHAUVIN Rose

Octobre 2025
PHP / MySQL / JavaScript


Lors de ce projet, nous avons créé un site web dynamique utilisant des données open source de la SNCF pour afficher des informations sur les gares, les horaires et les itinéraires des trains.

Ce projet m'a permis de développer mes compétences en développement web, en gestion de bases de données et en intégration de données externes dans une application web.

Organisation du projet

Lors de notre projet, nous avons commencé par analyser les données open source fournies par la SNCF, qui comprenaient des informations sur les gares, les horaires et les itinéraires des trains. Nous avons ensuite conçu une base de données MySQL pour stocker ces informations de manière structurée.

Nous avons ensuite développé le site web en utilisant PHP pour le backend et JavaScript pour le frontend, en intégrant les données de la base de données pour afficher les informations pertinentes aux utilisateurs. Nous avons aussi fait une gestion des utilisateur.
