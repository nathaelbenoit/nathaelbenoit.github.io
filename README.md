# Portfolio Nathaël BENOIT

Portfolio personnel présentant mon parcours en BUT Science des données à l'Université de Poitiers.

## Design

Design moderne et épuré inspiré de portfolios professionnels avec :
- Navigation sticky one-page
- Sections clairement définies
- Gradient coloré pour le hero
- Cards pour les projets
- Design 100% responsive

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Variables CSS)
- JavaScript vanilla
- Font Awesome (icônes)

## Structure

- **Accueil** : Hero section avec présentation
- **À propos** : Formation, compétences et passions
- **Projets** : Liens vers les projets par année
- **Bilan** : Bilan personnel et objectifs
- **Contact** : Informations de contact

## Démarrage

Ouvrez simplement `index.html` dans un navigateur ou utilisez un serveur local :

```bash
python -m http.server 8000
```

Puis ouvrez http://localhost:8000

## Pages

- `index.html` - Page principale (one-page)
- `pages/BUT1.html` - Projets 1ère année
- `pages/BUT2.html` - Projets 2ème année
- `pages/BUT3.html` - Projets 3ème année
- `pages/ExpPro.html` - Expériences professionnelles
- `pages/Bilan.html` - Bilan détaillé
- `pages/Contact.html` - Formulaire de contact
